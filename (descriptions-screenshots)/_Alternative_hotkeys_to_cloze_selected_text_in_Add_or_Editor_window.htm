2074653746 It is a very simple patch to create additional hotkeys in Add and Editor window Ctrl+Space which cloze selected text with maximum+1 number and Ctrl+Alt+Space which cloze selected text with maximum number. New keys are synonyms for old keys Ctrl+Shift+C and Ctrl+Alt+Shift+C respectively. You can invent your own keys combinations, specify them at the beginning of the source code and restart Anki.
