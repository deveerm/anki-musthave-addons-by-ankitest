1040866511 After importing a deck do you see that some cards show the target language on the front and some on the back? There is a way to modify a single card so the front and back are inverted:
  1. install this • Swap add-on
  2. and use F12 in Card Reviewer
You can easily add your own field name pairs in existing list (Tools - Add-ons - _Swap - Edit... - Save - Restart Anki): Pairs higher in the list take precedence over lower if some of them exist in the same note simultaneously. Field names in list are case sensitive by default, but user can define case insensitivity. To do so go to Tools ⇒ Add-ons ⇒ _Swap ⇒ Edit... and set up CASE_SENSITIVE = False then save the changes and restart Anki. You can swap fields also in Editor (and Add note) window, not only in Card Reviewer. There are a Sw button and F12 key for it. Date of swapping may be added as a hierarchical tag swapped::swap-2016-04-29 To activate tagging go to Tools ⇒ Add-ons ⇒ _Swap ⇒ Edit... and remove # sharp on line 25 then save the changes and restart Anki. To get exactly sw-16-04-29 tag uncomment line 26 (simply delete # sharp, just remove, not replace with space char).
