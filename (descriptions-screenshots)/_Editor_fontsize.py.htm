1931469441 Increase Text Size for Card Types Editor Would you like to change the size of text that appears in the Front Template, Styling, and Back Template text boxes? Then this add-on is for you! Solution out of the box. Just install, restart Anki and go on! Now it works with Edit HTML, Add-ons Edit... and Error Message too. F4 acts as well as Ctrl+Shift+X Position, width and height of HTML Editor and Add-ons Edit... dialog windows are remembered. You can change the typeface and font size as you need for each area separately. Just goto Tools - Add-ons - _Editor_fontsize - Edit... and edit FONTS list. Add/Edit Fields dialog is customizable too. Also in browser: Expands and Collapses all items in Browser Tree. Creates 3 options in Go menu to expand and collapse all items (Favourites searches, Decks and Tags) in Browse window.
  • Just press the Ctrl+Shift+Plus to expand,
  • Ctrl+Shift+Minus keys to collapse all and then expand first level sub-items
  • and Ctrl+Alt+Shift+Minus keys to collapse at all.
Or you can assign your own keyboard shortcuts.
