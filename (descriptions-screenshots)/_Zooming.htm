1071179937 Your cards look like too small? That's the add-on for you! The default font size is so big in Anki? That's 4U! Zooms, unzooms, lets you set a 1:1 (100%) or initial (user defined) zoom level. It's pretty cool. Work with images as well as text. Functions:
  • Zoom in or out with Ctrl++/Ctrl+-
  • or with Ctrl+mouse wheel
  • or with the View/Zoom submenu.
It just makes the font and images a bigger size. U can use a bigger font or scale images if you want without this addon. But you don't need to adjust images and the font size for meeting different resolution monitor each time any more. Improved:
  • remember user choice (also between sessions) — for each profile separately!
  • step ±10% (not */1.19)
  • info about current values Alt+0
There are three different multipliers:
  1. for decks panel
  2. for deck overview
  3. for cards
For each, there can be an user defined zoom factor set in the add-on file. Use Tools - Add-ons - _Zooming menu to edit this. Zoom Images simultaneously, not only text. Images can't be zoomed more than { max-width: 95%; max-height: 95%; } text can be zoomed min 0.1 with no max limitations. To avoid upper limit on images use in Styling section of your cards: img { max-width: none; max-height: none; } If you need old behaviour (zoom text only), goto Tools ⇒ Add-ons ⇒ _Zooming ⇒ Edit... and set ZOOM_IMAGES = False then save the changes and restart Anki. Essential! For small or huge screens - should be in by default. You don't need to zoom again in every new profile. The zoom level can be saved. Go to: Tools → Add-ons → _Zooming → Edit... Then change: deck_browser_standard_zoom overview_standard_zoom and review_standard_zoom each to for example 2.5 or 0.5 then save the changes and restart Anki. Reply buttons in reviewer and field names in editor becomes slightly more bigger.
